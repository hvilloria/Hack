## Commands
----
**install**
```
npm install
```
Dependency module install

**build**
```
npm run build
```
Do some magic with ES6 to create ES5 code.

**Run develop Server**
```
npm start
```

**clean**
```
rm -f dist/*
```

Delete existing dist files
You can see result in

OS X : http://0.0.0.0:8080

Linux : http://localhost:8080

Windows : http://localhost:8080
